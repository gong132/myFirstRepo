git 添加文件用vi命令加文件名，会打开一个vi编辑器在里面添加文件内容
git add +文件名，会把文件提交到暂存区，
git commit -m "文件修改的注释信息"，将文件提交到当前分支

git diff + 文件名 会显示修改的信息
git log 用来查看修改的日志
git reset --hard HEAD^回到上一个版本，
cat 文件名，会查看文件的内容
git reset --hard + 版本号，会回到那一个版本
//*****************************************************************************
vi 在命令行输入i，会进入编辑模式，按下esc会回到命令行模式 ：wq是保存并退出（需要在英文输入法下）
